Problem — students and job seekers were losing hours manually formatting resumes, and many strong candidates were getting filtered out by ATS bots before a human ever saw their application.

My role — led the Gemini API integration and built the interactive UI, working in a team of three.

Approach — streamed structured user input through the Gemini API to auto-generate a formatted resume, added an ATS scoring engine to flag keyword gaps, and wired up one-click PDF export.

Outcome — full resume generation in under 3 seconds with instant ATS feedback, plus a 40% faster initial page load after adding lazy loading and code splitting.
